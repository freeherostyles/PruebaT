# Fase 4: Frontend Base

## Objetivo

Construir la base visual y de navegacion del frontend para soportar autenticacion, layout principal y dashboard.

## Alcance

- login page
- rutas publicas y privadas
- bootstrap de sesion
- layout principal
- dashboard inicial con metricas

## Resultado

- flujo de login funcional
- persistencia de sesion con Zustand
- layout con drawer y app bar
- dashboard con estadisticas de proveedores
